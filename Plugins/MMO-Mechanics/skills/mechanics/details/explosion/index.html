Explosion Mechanic

Spawns block-breaking explosions or client-side cosmetic visual explosions.

Options​

  • yield: Configuration parameter.
  • break: Configuration parameter.
  • y: Configuration parameter.
  • fire: Configuration parameter.
  • breakblocks: Configuration parameter.
  • force: Configuration parameter.
  • f: Configuration parameter.

Example Configuration​

mechanics: [
  "explosion{yield=value,break=value,y=value,fire=value,breakblocks=value,force=value,f=value}"
]
