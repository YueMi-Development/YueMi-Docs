# Themes
YueMi Spigot Allowing To Customize Themes.

## Config Preview
```yaml
theme:
  mainColor: '&b'
  secondaryColor: '&f'
  titleColor: '&3&l'
  spigotName: YueMi
```

### Field - Themes
- **mainColor**<br />
Primary accent color.
> default: `&b`

- **secondaryColor**<br />
Secondary/default text color.
> default: `&f`

- **titleColor**<br />
Color for UI titles and headers.
> default: `&3&l`

- **spigotName**<br />
The display name of your server fork.
> default: `YueMi`
