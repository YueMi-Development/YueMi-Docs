# Mobs
Customize Mobs Related Features.

## Config Preview
```yaml
mob:
  entityCollisions: false
  mobAI: false
  optimizeMobs: true
```

### Fields - Mobs
- **entityCollisions**<br />
  Disables entity-to-entity collision calculations.
  > default: `false`

- **mobAI**<br />
  Disables AI pathfinding and logic for mobs.
  > default: `false`

- **optimizeMobs**<br />
  Enables various internal performance improvements.
  > default: `true`
