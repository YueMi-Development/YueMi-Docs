# Player
Customize Player Related Features.

## Config Preview
```yaml
player:
  disableFootSteps: true
  disableSwimSounds: true
  disableNpcSprintingSpeed: true
```

### Fields - Player
- **disableFootSteps**<br />
  Disables Foot steps effect for Player.
  > default: `true`

- **disableSwimSounds**<br />
  Disables Swim effects for Player.
  > default: `true`

- **disableNpcSprintingSpeed**<br />
  Disable NPC Sprinting Movement speed, forcing using walk speed.
  > default: `true`
