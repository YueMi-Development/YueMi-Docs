# Miscellaneous
Customize miscellaneous features and limiters.

## Config Preview
```yaml
misc:
  mace-damage-limit: -1.0
```

### Fields - Miscellaneous
- **mace-damage-limit**<br />
  Sets a limit on the maximum damage a Mace can deal. 
  > default: `-1.0` (no limit/disabled)
