# NotLivingNearOrigin Targeter

Targets non-living entities (like items or armor stands) within a radius around the origin.

## Shorthands & Aliases

- `notlivingnearorigin`
- `nonlivingnearorigin`
- `nlno`

## Options

- `r / radius`: The radius to search within (default: 5.0).

## Example Configuration

```json5
targeter: "notlivingnearorigin{radius=8}"
```
