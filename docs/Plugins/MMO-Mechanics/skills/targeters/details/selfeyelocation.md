# SelfEyeLocation Targeter

Targets the eye location of the caster.

## Shorthands & Aliases

- `selfeyelocation`
- `eyedirection`
- `castereyelocation`
- `bosseyelocation`
- `mobeyelocation`

This targeter does not accept any extra option parameters.

## Example Configuration

```json5
targeter: "selfeyelocation"
```
