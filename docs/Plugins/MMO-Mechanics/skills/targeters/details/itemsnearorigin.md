# ItemsInRadius / ItemsNearOrigin Targeter

Targets dropped items on the ground within a specified radius.

## Shorthands & Aliases

- `itemsnearorigin`
- `itemsinradius`
- `iir`

## Options

- `r / radius`: The search radius (default: 5.0).

## Example Configuration

```json5
targeter: "itemsnearorigin{r=6}"
```
