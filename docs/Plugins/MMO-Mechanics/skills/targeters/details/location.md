# Location / Origin / Source Targeter

Targets a specific set of coordinates (X, Y, Z) in the world.

## Shorthands & Aliases

- `location`
- `origin`
- `source`

## Options

- `x`: X coordinate value (default: 0.0).
- `y`: Y coordinate value (default: 0.0).
- `z`: Z coordinate value (default: 0.0).

## Example Configuration

```json5
targeter: "location{x=100,y=64,z=-250}"
```
