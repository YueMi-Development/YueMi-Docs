# Posearmorstand Mechanic

Sets specific rotations (head, body, arms, legs) on targeted armor stands.

## Options

- `rightleg`: Configuration parameter.
- `body`: Configuration parameter.
- `rarm`: Configuration parameter.
- `leftleg`: Configuration parameter.
- `rightarm`: Configuration parameter.
- `larm`: Configuration parameter.
- `head`: Configuration parameter.
- `leftarm`: Configuration parameter.
- `rleg`: Configuration parameter.
- `lleg`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "posearmorstand{rightleg=value,body=value,rarm=value,leftleg=value,rightarm=value,larm=value,head=value,leftarm=value,rleg=value,lleg=value}"
]
```
