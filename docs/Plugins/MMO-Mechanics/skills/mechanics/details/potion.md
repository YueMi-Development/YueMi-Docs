# Potion Mechanic

Applies potion effects with custom amplifier/duration, or clears active potion effects.

## Options

- `level`: Configuration parameter.
- `duration`: Configuration parameter.
- `icon`: Configuration parameter.
- `t`: Configuration parameter.
- `particles`: Configuration parameter.
- `type`: Configuration parameter.
- `d`: Configuration parameter.
- `amplifier`: Configuration parameter.
- `a`: Configuration parameter.
- `ambient`: Configuration parameter.
- `l`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "potion{level=value,duration=value,icon=value,t=value,particles=value,type=value,d=value,amplifier=value,a=value,ambient=value,l=value}"
]
```
