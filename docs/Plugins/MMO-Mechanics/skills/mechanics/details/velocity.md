# Velocity Mechanic

Adjusts entity speed and velocity vectors, including relative pushes/throws.

## Options

- `m`: Configuration parameter.
- `mode`: Configuration parameter.
- `velocity`: Configuration parameter.
- `x`: Configuration parameter.
- `y`: Configuration parameter.
- `v`: Configuration parameter.
- `force`: Configuration parameter.
- `f`: Configuration parameter.
- `z`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "velocity{m=value,mode=value,velocity=value,x=value,y=value,v=value,force=value,f=value,z=value}"
]
```
