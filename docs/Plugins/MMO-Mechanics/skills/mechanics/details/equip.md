# Equip Mechanic

Forces targeted entities to equip items or copies equipment configurations.

## Options

- `material`: Configuration parameter.
- `item`: Configuration parameter.
- `m`: Configuration parameter.
- `slot`: Configuration parameter.
- `a`: Configuration parameter.
- `amount`: Configuration parameter.
- `s`: Configuration parameter.
- `i`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "equip{material=value,item=value,m=value,slot=value,a=value,amount=value,s=value,i=value}"
]
```
