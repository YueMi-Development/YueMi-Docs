# Time Mechanic

Sets absolute or relative world times.

## Options

- `relative`: Configuration parameter.
- `r`: Configuration parameter.
- `t`: Configuration parameter.
- `v`: Configuration parameter.
- `value`: Configuration parameter.
- `time`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "time{relative=value,r=value,t=value,v=value,value=value,time=value}"
]
```
