# Animatearmorstand Mechanic

Animates armor stands by rotating their limbs over a duration of ticks.

## Options

- `rot`: Configuration parameter.
- `duration`: Configuration parameter.
- `rotation`: Configuration parameter.
- `t`: Configuration parameter.
- `limb`: Configuration parameter.
- `ticks`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "animatearmorstand{rot=value,duration=value,rotation=value,t=value,limb=value,ticks=value}"
]
```
