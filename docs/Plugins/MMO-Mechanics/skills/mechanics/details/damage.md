# Damage Mechanic

Deals damage to targeted entities as raw values, multiplier of caster base damage, or percentage of target health.

## Options

- `ignorearmor`: Configuration parameter.
- `ia`: Configuration parameter.
- `v`: Configuration parameter.
- `a`: Configuration parameter.
- `p`: Configuration parameter.
- `amount`: Configuration parameter.
- `percent`: Configuration parameter.
- `value`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "damage{ignorearmor=value,ia=value,v=value,a=value,p=value,amount=value,percent=value,value=value}"
]
```
