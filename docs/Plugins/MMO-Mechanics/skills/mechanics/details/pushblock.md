# Pushblock Mechanic

Physically slides blocks in a chosen direction (cardinal or facing direction).

## Options

- `distance`: Configuration parameter.
- `d`: Configuration parameter.
- `dist`: Configuration parameter.
- `dir`: Configuration parameter.
- `direction`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "pushblock{distance=value,d=value,dist=value,dir=value,direction=value}"
]
```
