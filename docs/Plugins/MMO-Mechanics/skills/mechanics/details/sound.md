# Sound Mechanic

Plays sound effects at locations or stops sounds for targeted players.

## Options

- `sound`: Configuration parameter.
- `volume`: Configuration parameter.
- `v`: Configuration parameter.
- `p`: Configuration parameter.
- `pitch`: Configuration parameter.
- `s`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "sound{sound=value,volume=value,v=value,p=value,pitch=value,s=value}"
]
```
