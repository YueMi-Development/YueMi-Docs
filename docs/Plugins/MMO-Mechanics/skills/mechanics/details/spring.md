# Spring Mechanic

Spawns temporary liquid sources (water or lava) that vanish after a duration.

## Options

- `duration`: Configuration parameter.
- `t`: Configuration parameter.
- `ticks`: Configuration parameter.
- `type`: Configuration parameter.
- `liquid`: Configuration parameter.
- `l`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "spring{duration=value,t=value,ticks=value,type=value,liquid=value,l=value}"
]
```
