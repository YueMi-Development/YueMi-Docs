# Sendtitle Mechanic

Displays title and subtitle text overlays on player screens.

## Options

- `fadeoutticks`: Configuration parameter.
- `fi`: Configuration parameter.
- `stayticks`: Configuration parameter.
- `st`: Configuration parameter.
- `fo`: Configuration parameter.
- `stay`: Configuration parameter.
- `t`: Configuration parameter.
- `subtitle`: Configuration parameter.
- `fadein`: Configuration parameter.
- `fadeinticks`: Configuration parameter.
- `title`: Configuration parameter.
- `s`: Configuration parameter.
- `fadeout`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "sendtitle{fadeoutticks=value,fi=value,stayticks=value,st=value,fo=value,stay=value,t=value,subtitle=value,fadein=value,fadeinticks=value,title=value,s=value,fadeout=value}"
]
```
