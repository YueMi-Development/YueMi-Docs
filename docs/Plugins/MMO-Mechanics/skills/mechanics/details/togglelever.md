# Togglelever Mechanic

Toggles levers, or triggers buttons to power on and automatically shut off after a delay.

## Options

- `power`: Configuration parameter.
- `duration`: Configuration parameter.
- `t`: Configuration parameter.
- `ticks`: Configuration parameter.
- `p`: Configuration parameter.
- `powered`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "togglelever{power=value,duration=value,t=value,ticks=value,p=value,powered=value}"
]
```
