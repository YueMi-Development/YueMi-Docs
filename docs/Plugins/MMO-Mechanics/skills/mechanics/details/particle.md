# Particle Mechanic

Spawns visual shapes (point particles, circles, ring outlines, box borders) around targets.

## Options

- `n`: Configuration parameter.
- `hs`: Configuration parameter.
- `vspread`: Configuration parameter.
- `r`: Configuration parameter.
- `vs`: Configuration parameter.
- `radius`: Configuration parameter.
- `particle`: Configuration parameter.
- `name`: Configuration parameter.
- `size`: Configuration parameter.
- `a`: Configuration parameter.
- `sz`: Configuration parameter.
- `amount`: Configuration parameter.
- `hspread`: Configuration parameter.
- `p`: Configuration parameter.
- `speed`: Configuration parameter.
- `s`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "particle{n=value,hs=value,vspread=value,r=value,vs=value,radius=value,particle=value,name=value,size=value,a=value,sz=value,amount=value,hspread=value,p=value,speed=value,s=value}"
]
```
