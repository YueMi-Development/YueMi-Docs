# Setblocktype Mechanic

Changes the type/material of blocks at target locations with optional physics updates.

## Options

- `material`: Configuration parameter.
- `m`: Configuration parameter.
- `t`: Configuration parameter.
- `type`: Configuration parameter.
- `p`: Configuration parameter.
- `physics`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "setblocktype{material=value,m=value,t=value,type=value,p=value,physics=value}"
]
```
