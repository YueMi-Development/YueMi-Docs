# Barcreate Mechanic

Shows, removes, or modifies progress bars on top of player screens.

## Options

- `overlay`: Configuration parameter.
- `color`: Configuration parameter.
- `id`: Configuration parameter.
- `t`: Configuration parameter.
- `name`: Configuration parameter.
- `v`: Configuration parameter.
- `style`: Configuration parameter.
- `barid`: Configuration parameter.
- `c`: Configuration parameter.
- `p`: Configuration parameter.
- `value`: Configuration parameter.
- `title`: Configuration parameter.
- `s`: Configuration parameter.
- `progress`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "barcreate{overlay=value,color=value,id=value,t=value,name=value,v=value,style=value,barid=value,c=value,p=value,value=value,title=value,s=value,progress=value}"
]
```
