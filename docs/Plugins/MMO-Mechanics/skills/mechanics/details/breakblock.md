# Breakblock Mechanic

Breaks the block at the target location, with options to drop items naturally or give them to the player.

## Options

- `item`: Configuration parameter.
- `give`: Configuration parameter.
- `d`: Configuration parameter.
- `a`: Configuration parameter.
- `g`: Configuration parameter.
- `amount`: Configuration parameter.
- `drop`: Configuration parameter.
- `i`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "breakblock{item=value,give=value,d=value,a=value,g=value,amount=value,drop=value,i=value}"
]
```
