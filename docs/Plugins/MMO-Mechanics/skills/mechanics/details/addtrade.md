# Addtrade Mechanic

Appends custom recipes/trades to villagers.

## Options

- `ing1`: Configuration parameter.
- `ingredient2amount`: Configuration parameter.
- `ingredient1`: Configuration parameter.
- `i1`: Configuration parameter.
- `r`: Configuration parameter.
- `i1a`: Configuration parameter.
- `ingredient2`: Configuration parameter.
- `ing2a`: Configuration parameter.
- `a`: Configuration parameter.
- `i2a`: Configuration parameter.
- `result`: Configuration parameter.
- `amount`: Configuration parameter.
- `ingredient1amount`: Configuration parameter.
- `ing1a`: Configuration parameter.
- `ing2`: Configuration parameter.
- `i2`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "addtrade{ing1=value,ingredient2amount=value,ingredient1=value,i1=value,r=value,i1a=value,ingredient2=value,ing2a=value,a=value,i2a=value,result=value,amount=value,ingredient1amount=value,ing1a=value,ing2=value,i2=value}"
]
```
