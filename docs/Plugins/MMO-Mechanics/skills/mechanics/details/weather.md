# Weather Mechanic

Modifies world weather (Clear, Rain, Storm) and duration.

## Options

- `weather`: Configuration parameter.
- `w`: Configuration parameter.
- `duration`: Configuration parameter.
- `t`: Configuration parameter.
- `ticks`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "weather{weather=value,w=value,duration=value,t=value,ticks=value}"
]
```
