# Blockmask Mechanic

Temporarily masks or unmasks blocks client-side, showing fake block materials to players.

## Options

- `material`: Configuration parameter.
- `t`: Configuration parameter.
- `type`: Configuration parameter.
- `m`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "blockmask{material=value,t=value,type=value,m=value}"
]
```
