# Fillchest Mechanic

Fills targeted containers (chests, barrels, shulker boxes) with items, optionally clearing them first.

## Options

- `material`: Configuration parameter.
- `item`: Configuration parameter.
- `clear`: Configuration parameter.
- `m`: Configuration parameter.
- `a`: Configuration parameter.
- `amount`: Configuration parameter.
- `i`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "fillchest{material=value,item=value,clear=value,m=value,a=value,amount=value,i=value}"
]
```
