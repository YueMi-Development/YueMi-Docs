# Heal Mechanic

Restores health to target entities by raw values or max health percentages.

## Options

- `v`: Configuration parameter.
- `a`: Configuration parameter.
- `p`: Configuration parameter.
- `amount`: Configuration parameter.
- `percent`: Configuration parameter.
- `value`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "heal{v=value,a=value,p=value,amount=value,percent=value,value=value}"
]
```
