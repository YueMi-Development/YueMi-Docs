# Setname Mechanic

Sets a custom name visible above targeted entities with full text coloring and placeholders.

## Options

- `n`: Configuration parameter.
- `display`: Configuration parameter.
- `show`: Configuration parameter.
- `visible`: Configuration parameter.
- `name`: Configuration parameter.
- `v`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "setname{n=value,display=value,show=value,visible=value,name=value,v=value}"
]
```
