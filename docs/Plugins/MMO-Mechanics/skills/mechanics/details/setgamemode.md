# Setgamemode Mechanic

Sets player game modes (Survival, Creative, Adventure, Spectator).

## Options

- `g`: Configuration parameter.
- `mode`: Configuration parameter.
- `gamemode`: Configuration parameter.
- `m`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "setgamemode{g=value,mode=value,gamemode=value,m=value}"
]
```
