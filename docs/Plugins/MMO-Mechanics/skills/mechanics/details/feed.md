# Feed Mechanic

Feeds targeted players, increasing food and saturation levels.

## Options

- `sat`: Configuration parameter.
- `v`: Configuration parameter.
- `a`: Configuration parameter.
- `amount`: Configuration parameter.
- `value`: Configuration parameter.
- `saturation`: Configuration parameter.
- `s`: Configuration parameter.

## Example Configuration

```json5
mechanics: [
  "feed{sat=value,v=value,a=value,amount=value,value=value,saturation=value,s=value}"
]
```
