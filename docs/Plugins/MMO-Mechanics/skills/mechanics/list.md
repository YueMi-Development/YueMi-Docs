# Skill Mechanics

Mechanics execute actions on the targets resolved by targeters. Click on any mechanic name to see options and details.

---

## Supported Mechanics

| Mechanic | Aliases / Shorthands | Description |
| :--- | :--- | :--- |
| **[Delay](./details/delay.md)** | `delay` | Pauses the skill pipeline execution for a specified duration (ticks or seconds). |
| **[Thunder / Lightning](./details/thunder.md)** | `thunder`, `lightning` | Strikes a visual and damaging lightning bolt at the targeted location/entity. |
