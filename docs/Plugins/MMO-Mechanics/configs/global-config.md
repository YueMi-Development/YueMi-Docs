# Global Configuration

The global configuration of MMO Mechanics is stored in the `config.yml` file.

## Default Configuration

```yaml
# Internal version tracker used for automatic migrations. Do not modify.
config-version: 2

hooks:
  # Enable or disable PlaceholderAPI integration (Default: true)
  placeholder-api: true

features:
  # Enable or disable the administrator command /mmomechanics (Default: true)
  command-enabled: true
```
