# Introduction

Villager Tweaks is a lightweight and modern Minecraft server plugin built for PaperMC. It provides custom tweaks to villager behaviors, including configuration versioning, sequential migration mechanics, and customized gameplay features.

## Tweaks

| Tweak | Description | Default Config Option |
| :--- | :--- | :--- |
| **Witch Curing** | Cures Witches back into Villagers when they are splashed with Weakness and fed a Golden Apple. | `Enabled` |
