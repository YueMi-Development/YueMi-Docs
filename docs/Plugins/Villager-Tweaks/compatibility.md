# Compatible Platforms

Villager Tweaks is designed for high-performance servers running PaperMC or compatible forks.

---

## Supported Servers

* **PaperMC** (1.21+)
  * *Usage*: Highly recommended. The plugin utilizes PaperMC's native YamlConfiguration comment APIs and particle/sound engines.
* **Purpur / Spigot** (1.21+)
  * *Usage*: Compatible, as long as they implement standard modern Bukkit API features.
